## Requirement 
#### Run project in docker container
1. Package code to Jar file with Maven.
2. Put jar file into docker image and run.
3. Test api locally.

可能要使用到的命令: 
- maven clean
- maven install
- docker run
- docker build
- docker start
- docker ps
- docker images
- docker rm
- docker rmi
- FROM
- COPY
- WORKDIR
- EXPOSE
- CMD
- ENTRYPOINT
