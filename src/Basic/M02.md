## Requirement 
### Implement api
**AC1：输入一个人名 Naruto，返回`hi, Naruto`**

该 API 的 Endpoint 定义如下：

|项目|说明|
|---|---|
|API Endpoint|*/api/greeting/say/{personname}*|
|HTTP Method|POST|

该 API 的返回结果定义如下：

|项目|说明|
|---|---|
|Status Code|200|
|Content-Type|*application/json*|
