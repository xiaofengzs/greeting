## Requirement 
#### Implement another api
**AC1：输入一个人名Tristan，返回`see you agian, Tristan`**


该 API 的 Endpoint 定义如下：

|项目|说明|
|---|---|
|API Endpoint|*/api/greeting/bye/{personname}*|
|HTTP Method|POST|

该 API 的返回结果定义如下：

|项目|说明|
|---|---|
|Status Code|202|
|Content-Type|*application/json*|
